cd [directory] - Change directory to the directory in '[]'.

git init - Initialize Directory for cloud
share.

git remote add [name] [remote-URL] - Add remote
location to the local branch. 

git add [file] - Add a file to the local
branch.If '.' then all changes are added. 

git commit -m [message] - Commit local changes
and prepare for upload. Message consists of 50
characters,discribing the main changes.

git push [remote (Name)] [branch] - uploading
to the "remote" location, on 'branch' named
branch. 

git fetch - Check for changes and download if
any (cache).

gti pull [remote (Name)] [branch] - Save 
changes to local brench.

git status - List all changes after latest
commit.

git brench [Name] - create new branch named
'Name'.

git checkout [Name] - Download a remote branch
/ switch branch.

git branch {--remote} - List branches, '--
remote' list remote branches.
