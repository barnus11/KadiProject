cd [directory] - Change directory to the directory in '[]'

git init - Initializing directory for cloud share

git remote add [name] [remote-URL] - Add remote location to the local branch.

git add [file] - Add a file to local the branch. If '.' then all changes are added.

git commit -m [message] - Commit local changes abd prepare for upload. Message consists of 50 characters, discribing the main changes.

git push [remote (Name)] [branch] - Upload to the "remote" location, on 'branch' named branch.

git fetch - Check for changes and download if any (cache)

git pull [remote (Name)] [branch] - Save changes to local branch

git status - List changes after latest commit.

git branch [Name] - Create new branch named 'Name'

git branch {--remote} - List branches, if '--remote' list remote branches.

git checkout [Name] - Download a remote branch / switch branch.
